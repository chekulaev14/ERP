ОБЯЗАТЕЛЬНО перед началом любой задачи (до чтения кода, до анализа, до любых действий):

1. Проверить подходящий Superpowers скилл через Skill tool. Даже если кажется что скилл не нужен — проверить. Если подходит хотя бы на 1% — вызвать.

Это НЕ опционально. Пропуск этого шага = нарушение workflow.

## Beads (bd) — только для крупных задач

Beads использовать только когда задача:
- займёт 3+ сессий
- содержит 5+ подзадач с зависимостями
- требует координации нескольких агентов
- является эпиком или крупной фичей (конструктор, рефакторинг архитектуры)

Для мелких задач (баг-фикс, добавить поле, поправить UI) — beads НЕ нужен. Просто делай работу.
