# Gorchev-V — Сервис управления производственными процессами

## Описание

Веб-сервис для управления производственными процессами в металлообработке. Пилотный проект — обкатка на одном клиенте, затем упаковка в отдельный продукт (агент по производству из проекта "Ваши Документики").

Этапы: закупка материалов → производство → техпроцесс → контроль количества.

## Первый этап — терминал для цеха

Рабочий авторизуется по PIN → выбирает изделие из каталога (категории с фото) → видит карточку изделия (галерея, описание, список комплектующих) → выбирает комплектующую, которую делал → указывает количество за смену → на основе этого считается сдельная зарплата.

Роли: рабочий, кладовщик, директор.
Склад: изделие = набор деталей (до 15). Остатки деталей, расчёт сколько изделий можно собрать, что докупать.
